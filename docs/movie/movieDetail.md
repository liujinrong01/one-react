
# 请求电影详细信息

## URL
http://v3.wufazhuce.com:8000/api/movie/detail/${id}

## 参数
电影id

## 请求示例
+ http://v3.wufazhuce.com:8000/api/movie/detail/145

## HTTP请求方式
GET

## 数据返回格式
JSON

##是否需要登录
否
